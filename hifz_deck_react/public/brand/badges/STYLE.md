# Hifzer Badge Style — “Playful Path”

Bright sticker-like seals for kids and families. Same circular shell family; chunkier marks and cheerful colors.

## Shell

- Thick white rim (~3px), soft colored drop shadow matching the fill
- Marks are bold silhouettes with optional sparkle dots / layered opacity
- Locked: cool gray + grayscale

## Do / Don’t

- Do: big shapes, clear metaphors (star, book, flame, cards, crown)
- Don’t: muddy teals, hairline detail, verse calligraphy, neon purple glow
